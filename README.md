# stellhub-foundation-java

StellHub Java Foundation SDK.

This SDK provides a pure Java unified access layer for:

- OpenTelemetry resource attributes
- Kubernetes and `STELLAR_*` environment variables
- W3C trace headers and `X-Stellar-*` headers via framework-neutral models
- Runtime context and request context
- Error semantic attribute mapping for logs, traces, and metrics

Spring and Spring Boot integration are provided by the `stellflux` repository.
